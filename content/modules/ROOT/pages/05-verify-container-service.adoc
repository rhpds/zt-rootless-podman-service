== Verify Container Service

We’ll check the service is running properly and then modify the
index.html through the persistent volume.

== Check the service

Run the following command to check that the httpd container is running.

[source,bash,run]
----
podman ps
----

.podman ps output
image::podman_ps.png[podman_ps.png]

Let’s check the systemd service is running the container.

[source,bash,run]
----
systemctl --user status httpd --no-pager
----

.systemctl status output
image::systemctl_status.png[systemctl_status.png]

You can stop and start the httpd service with the following commands.

To stop the httpd service.

[source,bash]
----
systemctl --user stop httpd
----

To start the httpd service.

[source,bash]
----
systemctl --user start httpd
----

Now let’s check that the web server actually works.

[source,bash,run]
----
curl http://localhost:8080
----

.curl output
image::curl_output.png[curl_output.png]
